# 💰 Expense Tracker

A simple expense tracking web app built with vanilla JavaScript.

## 🚀 Features
- Add expenses
- Delete expenses
- Total calculation
- Data persistence using localStorage

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## ▶️ How to Use
1. Open `index.html`
2. Add expense title + amount
3. View total spending

## 🌐 Future Improvements
- Categories
- Charts
- Monthly analytics
- Export to CSV
